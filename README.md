## 概述

Visual Studio Code 插件。

```
npm install -g @vscode/vsce
vsce package
vsce publish
```

### Pic

图床工具，支持以下图床，支持自定义名称。

- weibo
- qiniu
- tcyun
- upyun
- github
- aliyun
- imgur
- SM.MS
